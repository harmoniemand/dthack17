RODAR

Ein Tool zum automatisierten erfassen des baulichen Zustandes von Straßen und Verkehrssystemen.

RODAR steht dabei für ROad DAmage Report Wir wollen mit günstiger Technologie und vorhandenen Geräten Daten erfassen. Diese Daten werden mithilfe moderner KIs auf unseren Servern ausgewertet und aufbereitet. Mithilfe einfacher Schnittstellen und Karten können diese Daten dann weiter verwertet werden. So können z.B. Straßenbauämter auf notwendige Maßnahmen aufmerksam werden ohne selbst vor Ort zu sein.

RODAR - Box

Die RODAR-Box ist ein kleines Gerät, dass in Fahrzeugen von Partnern installiert wird. Partner sind z.B. Busunternehmen aus dem ÖPNV oder Taxiunternehmen. Die Box ist mit einem Accelerometer ausgestattet und erfasst die Beschleunigungsdaten. Peaks ind diesen Daten, die einen Schwellenwert überschreiten, werden an unsere Server übermittelt und mit einem Standort und einem Gerät verknüpft. Werden für einen Standort verstärkt Peaks festgestellt, gilt dieser als auffällig und wird auf der Karte markiert.

Erfasste Daten

Erfasst werden massive Ausschläge in der Beschleunigung der Geräte. Diese treten auf, wenn ein Fahrzeug über ein Hindernis auf der Fahrbahn fährt. Diese Hindernisse können auf Straßenschäden hindeuten. Der Wert des Ausschlages wird zusammen mit den GPS-Koordinaten, einer Device-ID und einem Zeitstempel im Portal gespeichert.
